About

This is a specialized variant of the QUIC Interop Runner. Usually it is used to test the interoperability of QUIC implementations. Here we exploint the great work of Marten Seemann to run satellite measurements on all of these client and server implementations.

The satellite link emulation uses this ns-3 scenario. Parameters are:

  • File size: 10 MB
  • RTT: 600 ms
  • Data Rate: 20/2 Mbps
  • Loss Rate (in SATLOSS test case): 1 %
